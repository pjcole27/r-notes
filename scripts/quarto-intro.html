Quarto template

This is a Quarto document. It mixes together Markdown text and code and can be rendered into a variety of formats such as HTML web pages and PDFs. There are three types of content. The text at the top of the document surrounded by three dashes is the YAML header, which controls details for the output of the document and settings that affect the whole document. The second type of content is Markdown text; the type of text you are reading right now. Any Markdown styling is open to you to use how you want. The third type of content is code chunks, which are open and closed by three back ticks (```).
